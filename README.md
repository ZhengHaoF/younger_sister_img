# younger_sister_img

#### 介绍
younger sister img是我正在维护的一个项目，正在更新中

本意是做个能分辨出图片优良的AI，分为3步
1. 通过爬虫收集大量图片目前大约有4万张
2. 通过网友评价为图片建立档案
3. 人工智能方面的开发
  
  现在处于第二阶段：
  通过爬虫min2.py获取网络上的大量图片，图片存放在腾讯云对象储存中

  并建立一个数据库来存放每张图片的评价

}
``
